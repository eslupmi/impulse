## Formating

`Inline code` used for:

  - environment variables (`IMPULSE_API_KEY`)
  - CLI commands (`impulse start`)
  - filenames (`impulse.yml`)
  - config keys (`alertmanager.url`)
  - config values (`true`, `false`)
  - HTTP status codes (`200`, `404`)
  - API endpoints (`/api/v1/incidents`)
  - keyboard symbols (`@`)
  - version numbers (`v1.3.2`)

**Bold text** used for:

  - important terms introduced in the text
  - headers
